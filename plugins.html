Logical Plugins!

Coming 2025

Here are some notes of how it will work!

  • Plugins will feature its own easy coding language with multiple keywords!
  • Will feature own plugins menu and commands
  • Coders will be able to hook in-game Rocket League function calls and manipulate what happens before or after!
  • Will be able to use *ANY* function in Logical Mod.
  • Will have the file extention ".lgl" (e.g. GiveAllPlugin.lgl)
